# NetMonitor

NetMonitor is a web-based network monitoring tool developed using Flask and Nmap.

## Features

- Detect devices in network
- Show IP address
- Show MAC address
- Show Manufacturer
- Professional dashboard

## Technologies Used

- Python
- Flask
- Nmap
- HTML
- CSS

## How to Run

Install dependencies:

pip install -r requirements.txt

Run application:

python app.py

Open browser:

http://127.0.0.1:5000
